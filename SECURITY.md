# Security Policy 2B
### Versions that apply
This policies are supported by all versions of **95Windows** from **2B** onwards.
### Data collection
No user data gets stored by the website while it's being used.
The only data that _intentionally and explicitly_ gets collected **according to user's choice** are the dark mode preference cookie and the `cookiesAccepted` cookie, both 100% optional.
Any security-releated issue can be reported by opening an [issue on this repo](https://github.com/jstmaxlol/95windows/issues)
